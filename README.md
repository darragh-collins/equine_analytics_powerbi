# Equine Analytics Dashboard Project

## Overview
This project is an end-to-end equine breeding analytics solution built in Power BI.

The goal of the project was to analyse:
- Stallion performance
- Mare breeding history
- Pairing success trends
- Foal success rates
- Breeder activity
- Covering volume across the market

The project simulates a real operational analytics environment within the equine breeding industry.

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- Photoshop
