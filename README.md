# Equine Analytics Project

## Overview

This project is an end-to-end equine analytics solution built in Power BI.

It uses several years of real Thoroughbred breeding data from Weatherbys to help buyers and breeders analyse stallion performance, mare breeding history and breeding pair success.

The report is built around practical business questions, such as:

- Which stallions consistently produce successful foals?
- Which mares have the strongest breeding history?
- Which stallion and mare pairings have performed best over time?
- How has foal success changed across recent breeding seasons?
- Which breeders and countries contribute the highest breeding volume?

The project uses Power Query for data cleaning, a star schema model for analysis, DAX measures for KPIs, and Power BI report pages designed around breeding and purchasing decisions.

## Business Context

This project was built using real breeding data purchased from Weatherbys, the official Thoroughbred registry for Britain and Ireland. The original data existed across multiple CSV files and required significant cleaning and modelling before it could be analysed effectively. The goal was to transform the raw breeding records into a practical reporting solution that supports breeding and purchasing decisions.

## Dashboard Pages

### Market Overview
![Market Overview](Market%20Overview.jpg)

The Market Overview page provides the overall context before analysing individual stallions or mares.

It presents headline KPIs, breeding activity, foal success trends, leading breeding countries and the most active breeders. The purpose of this page is to establish the overall market before drilling into individual breeding performance.

### Stallion Performance
![Stallion Performance](Stallion%20Performance.jpg)

The Stallion Performance page compares stallions across the entire dataset.

It highlights covering volume, live foals, foal success rate and the highest-performing stallions. This allows buyers and breeders to compare stallions using both breeding activity and breeding success before investigating an individual stallion in more detail.

### Stallion Lookup
![Stallion Lookup](Stallion%20Lookup.jpg)

The Stallion Lookup page provides a detailed view of an individual stallion.

Users can select a stallion and review its complete covering history, breeding records and foal success trend over time. This helps assess the long-term performance of a specific stallion before making breeding decisions.

### Mare Lookup
![Mare Lookup](Mare%20Lookup.jpg)

The Mare Lookup page focuses on an individual mare's breeding history.

It displays every recorded covering, the stallions used, breeding outcomes and foal success trends over time. The purpose of this page is to help users evaluate the breeding performance of a specific mare before selecting future pairings.

### Pairing Analysis
![Pairing Analysis](Pairing%20Analysis.jpg)

The Pairing Analysis page examines specific mare and stallion combinations.

It summarises previous breeding pairings, covering history, live foal outcomes and overall success rate. This allows users to identify combinations that have consistently produced strong breeding results.
